Process of Execution

Install arduino software from www.arduino.cc

Make the arduino setup ready by proper connection of ultrasonic distance sensor to the arduino board

Install Python

Install the following libraries in command prompt after installing python

1)open cv library

2)libraries for running google api

3)text to speech library

Run the arduino by uploading the arduino code with connected circuit

Type the command workon cv in terminal

Run the main_page.py python file in terminal by passing the arguements --prototxt MobileSSD_deploy.prototxt.txt and --model MobileNetSSD_deploy.caffemodel
